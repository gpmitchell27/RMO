This project was created by using iMovie. I only used iMovie one other time previous so there was a bit of a learning curve for me. I had to recreate a fullsail ad using various techniques taught throughout the month. The ease of use of the timeline made this project go alot smoother. Manipulating titles, transitions and even adding music was easy using iMovie.

The same technique used in most audio applications normalizing was also used to calibrate the decibal levels throughout my project. The most fun was the voiceovers that we did by using the ducking technique. I might have went a little overboard with mine, but it was all in fun.
